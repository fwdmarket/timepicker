Timepicker

A slick timepicker using MomentJs with default Twitter Bootstrap integration.

View project on GitHub

FwdMarket TimePicker

The goal of this project is to be an extensible time picker component.

The current version uses moment.js as a base for all time operations and Twitter Bootstrap dropdown component for displaying the time picker.

Examples

Check out some examples of the plugin in action: https://fwdmarket.github.com/timepicker/examples.html

Custom Design

The layout and design of the timepicker can be done by modifying the template from the TPGlobal.template variable and by altering the *.less files distributed with the archive.
